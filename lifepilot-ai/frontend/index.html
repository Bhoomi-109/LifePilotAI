          ℹ️ System Architecture & Kaggle Guide

          Under the hood of LifePilot AI: Multi-Agent workflows and Model Context Protocol integration.

          🔧 Kaggle Concept Checklist

          • Google ADK Multi-Agent System: The Coordinator Agent handles step-by-step logic, dispatching to specialized agents (Planner, Study, Scheduler, Optimizer).
          • Model Context Protocol (MCP) Server: Backend implements an official MCP server registering tools for calendar matching, task sorting, reminder compilation, time division, and burnout checks.
          • Input Sanitization: Express router validates incoming schemas. The AI helper filters override commands to secure the prompt loop.
          • Zero Secrets in Code: Secrets loaded dynamically via `.env` file, defaulting to mock mode for ease of local validation.

          📡 Active MCP Tools Schema

          These tools are fetched directly from the live MCP Server routes.
